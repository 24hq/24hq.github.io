What DevOps really is?

IT position / role

DevOps is a special guy in a team, usually referred as System Engineer, which is responsible for development tooling maintanence, improving development and ops communication, release management and much more

A set of practices

DevOps is a set of practices which aims to enhance the software delivery by creating tighter links between the different IT people including Developers and Operations through the use of automation and communication.
